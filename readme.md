# Berry Sickle

This mod dont have a Steam Workshop page since it was made for another modder. Will harvest Fiber and Berries from bushes.
[![SS](https://i.gyazo.com/b2157709c426acffea5de0d953eea839.jpg)](https://gyazo.com/68811b0b2a50b988b570d19c1f568103)

### Installation / Usage

Extract the files into your ARK mods folder, \ARKEditor\Projects\ShooterGame\Content\Mods\. 

You should not need to restart your kit, but its always recommended.
